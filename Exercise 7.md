

1. **Addition (`+`)**
    ```python
    result = 5 + 3  # result is 8
    ```

2. **Subtraction (`-`)**
    ```python
    result = 10 - 4  # result is 6
    ```

3. **Multiplication (`*`)**
    ```python
    result = 7 * 6  # result is 42
    ```

4. **Division (`/`)**
    ```python
    result = 20 / 5  # result is 4.0
    ```

5. **Floor Division (`//`)**
    ```python
    result = 20 // 3  # result is 6
    ```

6. **Modulus (`%`)**
    ```python
    result = 22 % 5  # result is 2
    ```

7. **Exponentiation (`**`)**
    ```python
    result = 3 ** 4  # result is 81
    ```
